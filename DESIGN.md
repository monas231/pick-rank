## 2. 주요 기능

### 랭킹
- 전체 랭킹 (맛 70% + 가성비 30%)
- 맛 랭킹 (5개 항목 가중 평균)
- 가격 랭킹 (100g당 가격 오름차순)
- 가성비 랭킹 (맛 점수 / 100g당 가격, 정규화)
- 품평회 랭킹 (aT 대한민국 김치품평회 수상작 기준)

### 김치 상세 페이지
- 오각형 레이더 차트 (5개 맛 항목)
- 가격 변동 그래프 (시계열)
- 현재 최저가 정보 (판매처, 링크, 구매방법)
- 사용자 리뷰 목록

### 사용자 기능
- 맛 평가 입력 (5개 항목, 1~10점, 1인 1리뷰 + 수정 가능)
- 가격 게시물 등록 (판매처, 가격, 용량, 링크 등)
- 리뷰 공감(좋아요)

### 관리자 기능
- 김치 상품 등록/관리
- 품평회 수상 결과 입력 (aT 공식 PDF 참고하여 수동 입력)

## 3. 맛 평가 항목 (5개 — 오각형 레이더 차트)

| # | 항목 | 설명 | 범위 |
|---|------|------|------|
| 1 | 매운맛 | 고춧가루 화끈한 정도 | 1~10 |
| 2 | 신맛 | 발효 숙성도 | 1~10 |

| 4 | 단맛 | 달달한 정도 | 1~10 |
| 5 | 아삭함 | 식감/신선도 | 1~10 |

> 감칠맛은 젓갈맛과 중복 가능성이 높아 제외

## 5. 랭킹 계산 로직

### 시간 기반 가중치 — 동적 구간

**Step 1. 구간 결정**
- 최근 3개월 리뷰 수 >= 5 → 3개월 구간 사용
- 최근 6개월 리뷰 수 >= 5 → 6개월 구간 사용
- 전체 리뷰 수 < 5 → 전체 기간 단순 평균

**Step 2. 구간 내 가중치**
- 최근 3개월 × 3.0
- 3~6개월 × 2.0
- 6개월~1년 × 1.0

### 맛 점수 계산 (시간가중치 + 랭커가중치 동시 적용)

```
맛 점수 = Σ(review_score × 시간가중치 × 랭커가중치)
         / Σ(시간가중치 × 랭커가중치)
```

### 랭킹별 계산

| 랭킹 | 계산식 |
|------|--------|
| 맛 점수 | (매운맛 + 신맛 + 젓갈맛 + 단맛 + 아삭함) / 5 (가중 평균 적용) |
| 가격 랭킹 | price_per_100g 오름차순 |
| 가성비 | 맛 점수 / price_per_100g (0~1 정규화 후 비교) |
| 전체 랭킹 | 맛 점수 × 0.7 + 가성비 점수 × 0.3 |
| 품평회 | competition_award 기준 (대상 > 금상 > 은상 > 동상) |

## 6. 랭커(Ranker) 시스템 — 신뢰도 기반 평가 가중치

### 6.1 개념

사용자의 기여도(리뷰, 가격 게시물)와 타인으로부터 받은 공감을 합산해 신뢰도 점수를 계산한다. 일정 점수 이상이면 "랭커" 등급을 부여하고, 랭커의 리뷰는 점수 계산 시 일반 사용자보다 더 큰 비중을 가진다.

별도의 추천 섹션이나 발굴자 표시는 두지 않는다 — 랭커가 후하게 평가한 신규 제품은 가중치 덕분에 기존 랭킹(전체/맛/가성비)에서 자연스럽게 상위로 노출된다.

### 6.2 신뢰도 점수 계산 (초안 — 가중치는 추후 조정)

| 행동 | 점수 |
|------|------|
| 리뷰 작성 | +2 |
| 가격 게시물 등록 | +1 |
| 내 리뷰가 받은 공감 1개당 | +1 |
| 내 가격 게시물이 받은 공감 1개당 | +1 |

```
reputation_score = 기여도 점수 합 + 공감 점수 합
```

### 6.3 등급(Tier)

| 점수 범위 | 등급 | 리뷰 가중치 |
|-----------|------|-------------|
| 0 ~ 99 | 일반 사용자 (general) | 1.0 |
| 100 ~ 299 | 신뢰 사용자 (trusted) | 2.0 |
| 300 이상 | 랭커 (ranker) | 4.0 |

> 랭커 등급은 프로필에 배지로 표시

### 6.4 미확정 사항 (구현 전 결정 필요)

- 점수 가중치(+2, 1.0/2.0/4.0 등)는 초안이며 실사용 데이터 확인 후 조정 필요
- 자기 콘텐츠에 자기가 공감 누르는 것을 점수에서 제외하는 로직(self-like 방지) 필요
- 랭커 등급 강등 조건 (활동 중단 시 등급 유지 여부) 미정
